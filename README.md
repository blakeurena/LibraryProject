# Library Project

A responsive library website showcasing book collections with modern design and interactive features.

## Project Structure

```
LibraryProject/
├── css/
│   └── library.css          # Stylesheet for the library website
├── js/
│   └── library.js           # JavaScript for menu interactions
├── libraryassets/
│   ├── Library.svg          # Logo
│   ├── Undraw_Books.svg     # Hero illustration
│   ├── wave.svg             # Decorative wave element
│   └── [book covers]        # Various book cover images
├── library.html             # Main HTML page
└── README.md                # This file
```

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Featured Books Section**: Highlights popular books including:
  - Crack the Coding Interview
  - Atomic Habits
  - Can't Hurt Me
  - Deep Work

- **Latest Books Collection**: Displays 8 recent additions to the library
- **Interactive Menu**: Mobile-friendly hamburger menu
- **Modern UI**: Clean purple theme with smooth animations

## Getting Started

1. Clone this repository
2. Open `library.html` in your web browser
3. Enjoy browsing the library!

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome icons
- Google Fonts (Roboto)
